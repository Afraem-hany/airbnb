# 🏠 Airbnb Dashboard Project – Tableau

## 📊 Overview

This project presents an **interactive dashboard** built with **Tableau**, using real Airbnb data. The goal is to explore key trends in listings, pricing, availability, and host performance across various locations.

The project follows a complete data workflow:
1. SQL analysis & data preparation  
2. Data cleaning & transformation  
3. Interactive visualization with Tableau

---

## 🎯 Objectives

- Build a professional, clean, and interactive dashboard in **Tableau**
- Derive insights about:
  - Room types & distribution
  - Price ranges & averages
  - Host activity & performance
  - Availability trends over time
- Practice real-world data visualization and storytelling

---

## 🛠️ Tools Used

- **SQL Server** – Data exploration and extraction  
- **Excel** – Data cleaning (optional intermediate step)  
- **Tableau Public** – Interactive dashboard creation
